# homework_211123_01

## description

輸入第一個整數，代表測試資料筆數，
接下來每行輸入兩個浮點數，使用數學符號fmod求浮點餘數。
輸入範例：
5
5.33 2.33
13.657 2.333
8.33 1.3
8.3 4.2
95.3 4.33
輸出範例：
0.670000
1.992000
0.530000
4.100000
0.040000
